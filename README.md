# Deep Learning - Playground Bölüm II

Bu egzersiz, sinir ağlarının farklı **hiperparametrelerinin** algoritmanın **yakınsamasını (convergence)** nasıl etkilediğini görsel olarak incelemek için **TensorFlow Playground** aracını kullanır.

Bu görselleştirme, Playground’da tanımladığınız sinir ağını **Keras** kütüphanesini kullanarak koda dökmenize yardımcı olmayı amaçlar.
